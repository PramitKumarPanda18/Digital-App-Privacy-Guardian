# Digital App Privacy Guardian 🔐

## Overview
Digital App Privacy Guardian is an AI-powered system that analyzes mobile app permissions and detects potential privacy risks.

The system helps users understand whether an application might misuse sensitive permissions.

---

## Problem Statement
Many mobile applications request unnecessary permissions that may compromise user privacy.

This project aims to detect risky apps by analyzing permission patterns using Machine Learning.

---

## Features
- Permission analysis
- Privacy risk detection
- AI-based risk scoring
- User-friendly dashboard
- App privacy insights

---

## How It Works
1. User uploads or selects an app.
2. System extracts requested permissions.
3. AI model analyzes permission patterns.
4. Privacy risk score is generated.
